# Kotlin version of the test

I only manage to create the application with 2 endpoints:

- `POST /activation`
    - This endpoint only get activation request with date and volume and return a Fake response
- `PUT /assets`
    - This endpoint is used to update the default assets of the application

## Running the application

The application to run `com.flexcity.flexcity_app_kotlin.FlexcityAppKotlinApplication`

## Test endpoints

There is a `requests.http` to test the endpoints
